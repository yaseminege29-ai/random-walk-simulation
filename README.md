# 2D Random Walk Simulation

This project simulates a 2D random walk and analyzes the behavior of the particle using Mean Squared Displacement (MSD).

## 📌 What is done?

- Simulated random motion of a particle in 2D
- Calculated Mean Squared Displacement (MSD)
- Averaged results over multiple runs
- Analyzed how MSD changes with time (steps)
- Compared MSD for different step counts

## 📊 Key Result

The results show that MSD increases approximately linearly with time, which is a characteristic behavior of diffusion processes.

## 🧠 Concepts Used

- Random Walk
- Brownian Motion
- Diffusion
- Statistical Averaging
